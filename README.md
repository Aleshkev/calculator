# Pocket Calculator

This is a fully functional pocket calculator implemented in React. You can [see it online here](https://aleshkev.github.io/calculator).

Tech stack:

- Vite
- Vitest
- React
- Tailwind

Features:

- [x] basic math operations
- [x] keyboard input
- [x] memory
- [x] responsive layout
- [x] unit tests
